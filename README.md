# NLW eSports - Trilha Ignite

Projeto desenvolvido na trilha Ignite durante a NLW eSports da Rocketseat.

## Tecnologias:

- Typescript

### Web
- React
- Tailwind CSS
- Vite
### Mobile
- React Native
- Expo
### Back-end
- Node.JS
- Express
- Prisma ( SQLite )
